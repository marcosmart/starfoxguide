# Lylat Briefing Room

An unofficial, fan-made strategy companion for **Star Fox (Nintendo Switch 2, 2026)**. Maximum-score strategies, medal hit targets, route planning and full Challenge Mode tactics for every stage in the game, wrapped in an interactive single-page site.

**Live site:** https://marcosmart.github.io/starfoxguide/

## Features

- **Briefing.** How scoring actually works: charged-shot bonus hits, hidden enemy spawns, ally kill stealing, checkpoint pacing, controller rapid-fire mapping and the medal rules, plus notes on the v1.1.0 patch changes that affect score chasing.
- **Interactive Lylat route map.** All sixteen stages drawn to match the in-game hologram, with four recommended routes (including the maximum-score Water Route) that light up across the star chart, and every branch unlock condition explained.
- **Stages and scoring.** Expandable cards for all sixteen stages with medal hit targets, checkpoint pace checks, exit triggers and stage-specific strategies.
- **Challenge Mode.** All 192 objectives (six Normal and six Expert per stage), each with a tactic.
- **Progress tracker.** Tick off challenges and medals; progress is saved in your browser via local storage and survives refreshes. Includes per-stage counters, an overall progress bar and a reset option.
- **Five themes.** Cockpit, Fox, Peppy, Falco and Slippy colour schemes, cycled from a single button and remembered between visits.
- **Fully responsive.** Designed to work on phones as well as desktop.

## How to use

Open the live site, no installation required. Everything runs client-side in a single HTML file with vanilla JavaScript, so you can also download `index.html` and open it locally in any modern browser.

Your checklist progress and theme choice are stored in your own browser's local storage. Nothing is sent anywhere, there are no accounts and no tracking. Clearing your browser data will clear your progress, and progress does not transfer between devices.

## Accuracy

Data is compiled from official patch notes, post-launch community guides and testing, current to game version 1.1.0 (13 July 2026). If a future patch changes hit requirements or challenge objectives, some figures may need updating, so verify anything critical in game.

## Copying and contributing

You are welcome to fork this repository and build your own version, that is what the licence is for. I am keeping this copy as my own, so pull requests will generally not be merged, but feel free to open an issue if you spot a factual error in the guide.
